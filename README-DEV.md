## 配置首页展示的分类

```yaml
修改 _config.yml
home:
  categories: ["BTC", "ETH", "CKB"]
```

## 配置文章必要信息

```md
---
title: 测试文章
date: "2021/2/13 20:46:25"
author: "阿剑"
avatar: "/images/avatar.png"
cover: "/images/cover.png"
categories:
- "ORGIN-OF-MONEY"
tags:
- a
- b-c
---

```

author: 作者
avatar: 作者头像
cover: 文章封面
categories: 文章分类
  空格用中划线('-')代替
tags: 文章标签
  空格用中划线('-')代替

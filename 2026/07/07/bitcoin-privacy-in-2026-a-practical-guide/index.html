比特币隐私性实用指南（2026）

Juan Galt2026-07-07
隐私性

作者：Juan Galt

来源：https://bitcoinmagazine.com/guides/bitcoin-privacy-in-2026-a-practical-guide

自比特币诞生以来，其隐私性理论和实践就在不断发展。虽然一度被说成是匿名的（anonymous），但其实最好把它当成是一个实用假名（pseudonymous）的货币系统。虽然比特币自身的运作不需要用户的个人信息，但围绕比特币的公司常常将用户的公钥 —— 也就是比特币世界里的账户 —— 跟 TA 的个人信息关联起来。他们这么做，有时候是为了满足法律上的金融行业监管要求，有时候，则是为了方便用户使用。

结果是，用户可能会分享或者暴露个人信息给这样的公司，比如自己的家用 IP 地址，这可能会被用来识别用户的互联网服务供应商，再凭此定位用户的家庭地址。真名、手机号码、收件地址，也都是一样的道理。所有这些信息，到了坏人手里，就会让当事人置于人身伤害和经济损失风险中。

值得强调的是，比特币自身并不是这些隐私性问题的根源，许多批评意见都误解了。现代社会固有隐私性问题，一直都没有解决， 导致人们的社会生活各方面的数据频繁泄露，从银行系统到社交网络，从政府机构到军队。我们日益依赖的电子社会常常无法保护我们的数据。

比特币与所有其他可以相互比较的制度都不同，它不需要采集用户的数据。实际上，它是在乎隐私性的个人可以接受的少数几种金融工具之一。现金是另一种，但其交易的距离有限，而且还有许多其它方面的缺点。

但是，作为一个电子系统，在登记用户信息（KYC）的交易所已成主流、现代软件公司的数据饥渴越演越烈的环境下，比特币是不是真的能隐私地用起来？这个问题的答案可能会吓到你。

隐私性，来自哪里？

在你生活的司法辖区或地方法律体系中，某些风险或者说威胁，会比其它类型的风险更紧迫。一些国家有时会对公民实施的资本管制，通常是直接在银行层面实施。比特币，如果以自主保管的方式持有，并搭配一定的隐私性习惯，可以保护用户免受此种威胁。

在另一些地方，国家倒是足够稳定，但有组织犯罪猖獗，导致有针对性的钓鱼活动甚至绑架，比如在法国，诚实而勤劳的个人为自己的密码货币资产交税，但这种地方法律的结果是，持有密码货币的情形进入了公开的记录，导致入室抢劫案件剧增。

最后，并非无足轻重的是，一些活动家可能身处压迫之下，无法使用银行并且与社会机构隔绝，以巧妙的方式使用比特币可能是他们唯一的经济依靠。取决于具体情形，一些工具和策略更能胜任。

隐私性并不意味着你不能成为一个守法的公民。许多国家都有健全的隐私性法律，意在保护公民免受各种威胁，同时让公民可以（举例而言）遵守税法。隐私性并不意味着你必有见不得人的事，希特勒的宣传部长戈培尔才会这么说隐私性。隐私性，是民主社会的基石。

网络隐私性

首先，你必须保护自己的 IP 地址，就是你的互联网服务商给你的设备（包括你的手机）安排的 ID 。处理这个问题最流行的方法是使用 “虚拟私人网络”。

并不是所有的虚拟私人网络都有一样的效果，有一些还被指控会保留以及出售你的数据。在开始之前，你需要深入研究一番，不能只看广告的一面之词，要问问你周围那些习惯疑神疑鬼的人。

在比特币圈子，Mullvad 的名声不错。而且他们接受比特币支付很长时间了，而且用起来很简单。 他们是跟 Tor 搭配使用的，可以阻绝所有不通过虚拟私人网络的流量。一个账户可以在多台设备上使用，也包括手机。

Tor 浏览器，也是一种趁手的重要工具。我们下面要说的许多隐私性工具都指出 Tor 连接，通常都内置了必要的软件库，所以你只需点几下按钮就能使用 Tor 网络。个人感受，开启了之后 app 会运行得慢一些，因为 Tor 需要运行一些匿名化计算，仅供参考。Brave 浏览器也值得一提，因为它会屏蔽绝大部分广告跟踪，并且内置了 Tor 支持。

私密地获得比特币

让比特币获得隐私性的最大挑战来自于用户积累比特币的环节。交易所、类似于经纪人的协助兑换比特币与法定货币的私人公司，已经成了最高效也最实际的购买比特币的方法。他们能够在敌意的法律制度、黑客团体和手长的执法机构下生存，常常是因为他们过度遵守了金融监管条例，而这些条例会要求他们收集用户的大量个人数据。

保护隐私性的替代方案，则反过来，常常被政府机构定期骚扰，通常无法生存，或者即使生存也无法在中心化的竞争对手面前保住市场份额。这种反差的一个著名案例是 “LocalBitcoins”，他们是第一家点对点将比特币换成法定货币的交易所，从 2013 年开始，运营了 10 年之后，最终关停。他们公司面临来自芬兰政府的越来越大的压力，被迫在 2019 年实施了 KYC， 最终在 2023 年的熊市和 Operation Chokepoint 2.0 行动中关门。

LocalBitcoins 连接买家和卖家，自身充当交易比特币的托管方，而法币是从买家的银行账户转到卖家的银行账户。LocalBitcoins 发明了这种模式，自身完全不接触法币，也不知道卖家的银行信息。只有在买家卖家发生争议时，这些信息才会传递给运营者。如果买家和卖家没有异议，BTC 就会直接由托管方释放给买家。

这种准去中心化的交易所模式，由 LocalBitcoins 始创，通常被称为 “点对点的比特币交易所”，虽然它有许许多多的变种，在隐私性上也各有取舍。

今天，Bisq.network 可能是 LocalBitcoins 的最著名的后继者。Bisq 尝试翻过 LocalBitcoins 的中心化缺点这一页，它创造了一种 Tor 匿名化的、去中心化的交易平台，允许全世界的比特币买家和卖家连接起来。Bisq 依然在运行，并且推出了多种方便的软件工具。用户可以在自己的机器上运行 Bisq 、在自己的手机上用 Bisq Connect 控制自己的账户，他们还能通过 Bisq Notifications 接收交易提醒。还有一种专用的移动端 app ，叫做 Bisq Easy 。

Bisq 的交易量大概在 每个月 500 万美元，从中心化交易所的标准来看是低的， 但对于普通用户的定期投资来说，已经够用了。在使用 Bisq 的时候，需要注意几件事：首先，你应该总是选择声誉较高的对手方。然后，你还应该注意他们收取的佣金，通常卖家会比现货价格多收 5% 。所以，你要找的是声誉最好、价格最便宜的对手方。Bisq Easy app 有很好的用户界面，并且给刚上手 P2P 买卖的新用户提供了很好的指导。

全世界还有很多这样的活跃的 P2P 交易所和平台。原则上，使用这些平台时，最好一次只买卖较少的数量，这样你才不会承担太多风险。当然，你的时间也有价值，所以数额也不该太小，不过，超过 10,000 美元的价值可能已经太多了。总的来说，定期投资策略，跟 P2P 囤币是天生一对。

另一种既能获得比特币、又具有很好的隐私性的办法是找到你居住地的比特币社区。许多大城市都有活跃的比特币社区。如果你住的地方还没有，那么你可以自己开一个比特币聚会，你可能会惊讶怎么会有这么多人出现。从这一点开始，慢慢跟本地的比特币人建立信任，也许你就能用现金买到一些 BTC 。许多比特币人是用比特币来接收工作报酬的，常常需要卖出一些来覆盖法币之处，所以有在真实世界里开展 P2P 交易的机会。

最后，还要再提一句，你可以用比特币来接收工作报酬， 建立一个新项目或者围绕比特币的品牌。这将给你处理自己的比特币收入信息的更大控制权。

区块链隐私性

不过，当你收到一些比特币之后，想要在间谍之眼下保护你的信息，就有许多办法了。比特币与在它之前的货币不同，是生存在一个公开的网络中的，比特币在这样的网络中转移的完整的交易历史，可以被任何人审计，只是它没有关联到持有者的个人信息，而使用公开的地址，或者说假名的比特币账户号码。

这些地址是公开的（在区块链数据库中），而数据公司可以尝试找出资金流动的线索，尤其是跟交易所在数据共享上合作，或者在进入公开领域的相关信息中寻找。用户可以通过使用多种工具和策略来保护自己免受这样的跟踪。

运行自己的节点

为了尽可能减少你向其分享自己的地址和余额的对象，也就是保护自己的隐私性，运行自己的比特币节点是非常重要的，否则，你就总是要 “问” 运行着节点的某个人，你的余额是多少。所有的钱包软件，但凡它不会在你的设备上直接运行一个比特币全节点，就必然依赖于运行在别处的比特币节点；又或者，它会将你的余额查询请求重新定向到某个公开的节点（由某些人出于公益或非公益理由而运行的公开节点）。

即使有了网络隐私性，比如你使用了私人虚拟网络，可以保护你免遭不运行自己的节点的风险， 但自我治理和隐私性下一步无疑是控制你所查询的节点，然后称为比特币网络的主动参与者，也就是运行自己的节点。

Sparrow Wallet 是一款非常流行的桌面端钱包软件，对隐私性特性、硬件签名器和高级的比特币特性比如多签名账户和 “静默支付” 都有绝佳的支持；他们提供了一份说明书，教你运行和使用自己的节点。他们的结论是，Fulcrum，作为 Bitcoin Core 软件的一个封装器，让区块链数据可以提供给用户的钱包软件，正是用户需要的东西。

作为一款桌面端钱包软件，Sparrow 可以在你的家庭网络内部运行（访问你自己的节点），这让你访问比特币区块链的行为具有更强的隐私性。如果你希望用自己的手机或笔记本电脑，在家庭网络之外访问你的节点，你需要在家庭网络中运行一个 Tor 隐藏服务（类似于 Tor 隧道），从而可以安全且隐私地远程访问它。

Boltz

Boltz 是一个交换比特币与其它密码货币的非托管交易所。它不涉及法币，并且永不持有用户的资金。在 Boltz 上，用户们使用一种叫做 “原子化互换” 的技术来相互交易，这意味着，交易中的任何一方都无需信任对方，互换的密码货币是几乎同一时间从一方发给另一方的。

Boltz 无需登记任何个人信息就能使用，并且可以通过 Tor 来访问，这让比特币用户可以利用其它区块链和支付网络的优势，并且有很强的隐私性。

可以通过 Boltz 来访问的一种网络是 “Liquid” 区块链，它是一个以比特币为价值单位的 “联盟化侧链”，带有强大的隐私性特性。另一个案例则是闪电网络，它有强大的潜在隐私性优势，因为它结算交易的机制不使用区块链网络，因此留下的公开记录将极其简单。可以利用 Boltz 将比特币换成稳定币和其它主流的区块链货币，从而比特币用户能够通过一个高度隐私的桥梁接触广大的密码货币产业以及它们的市场。

Boltz 可以通过他们的网站来使用，也可以下载一个单独的开源网页 app 。资深用户可以使用一种命令行界面，并且，因为整套技术是开源的，用户甚至可以自己运行一套 Boltz 设施来开展业务。因此，Boltz 消除了通过中心化交易所来切换区块链的需要，从而也消除了对应的隐私性风险。

Liquid Network

Liquid Network 是 Blockstream 公司创造的一个联盟区块链，正在慢慢变成比特币行业的一个重要的基础设施。 它在 2018 年启动，是比特币的一个有所修改的复刻；该网络的原生资产 LBTC ，是与 BTC 直接锚定的。要铸造 LBTC，你必须存入 BTC 到一个联盟的多签名钱包中；而要取回你的 BTC，你既可以选择 “脱锚”，也可以选择在许多原子化互换交易所将 LBTC 换成 BTC 。虽然 Liquid Network 的共识结构与比特币的不同，而且根本上是有准入门槛的，它站在全球数十家行业领先公司的肩膀上，而且从上线以来就一直非常稳定。

Liquid Network 的一个有趣之处就是其隐私性特性。Liquid Network 上的交易的数额和资产类型，都是默认加密的。在区块链上可以看到资产从 A 地址转移给了 B 地址，但转移的是哪种资产、转移了多少，是看不见的，只有相关参与者自己才能看见。它使用了一种就叫做 “机密交易” 的密码学技术，这是由 Adam Back、Andrew Poelstra、Mark Friedenbach、Gregory Maxwell 和 Pieter Wuille 发明的。Liquid 用起来也很便宜，并且出块时间更短（相较于比特币），这就使它成为了比特币隐私性工具包中的一个有趣工具， 尤其是搭配 Boltz 这样的隐私性桥梁。

BBlockstream 推出了一款移动端钱包，功能很强大也很容易使用，支持 Liquid Network 。

静默支付

静默支付是一种新型的比特币地址，具有完全不同的余额审计方式。

用户能看到各个地址（比特币账户号码）里面有多少 BTC 的全部意义在于，这样一来，用户就能容易地验证总供给以及比特币货币网络的经济完整性。而静默支付则让用户的收款标识符与比特币地址解耦。这种技术非常强大，而且在比特币行业中有长久的开发历史，在最近几年中获得了更多采用。

目前为止，能够收取静默支付的钱包软件比较少，Sparrow 钱包可能是其中最好的，支持各方面的隐私性特性，包括连接到用户自己的节点。静默支付地址可以重复使用，所以用户只需生成一个，就能一直使用，然后在自己的桌面电脑或笔记本电脑使用 Sparrow 来检查自己的余额。要获得额外的隐私性，用户可以搭配运行一个 Frigate 服务端， 它能够以自主保管的方式处理静默支付。

Payjoin

另一种能够填补空白的重要技术是 Payjoin 。有一个专门为开发它而云锁的基金会，而且钱包支持日益增加，这种简单的交易构造技术能够打破区块链分析常用的识别个体用户及其资金流动的线索。Sparrow 与多种钱包都支持 Payjoin ，因为它正成长为比特币支付的 HTTPS 。

Coinjoin

此种技术一度是比特币隐私性的核心， 支持 Coinjoin 的钱包软件（比如 Wasabi）让你可以跟其他人一起构造比特币交易，并且是非托管的。在处理得当时，这种技术拥有巨大的优势，并且今天仍有许多人在使用，虽然它也有一些棘手的缺点。Bitcoin Optech 的创办者和作者 Gustavo 说，“在我看来，Wasabi 比以往任何时候都更好用，而且是目前为止流动性最大的比特币隐私性解决方案。” 在 Coinjoin 中，交易的流动性池大小就是你得到的隐私性的大小。 “Kruw.io 是市场最大的协调员：它汇集了市场上 97% 流动性。” 并且，“每个月有大约 30,000 BTC 的交易量，以及价值 4000 BTC 的新输入。”

Coinjoin 变得如此有效而且流行，导致了标志性的 Samourai 钱包案件；这种钱包软件有一个自己的 coinjoin 实现。这是发生中的追求隐私性权利的文化斗争。

Gustavo 也列举了 Coinjoin 的一些缺点，比如中心化交易所也许能辨别出你的资金来自一笔 coinjoin 交易，因为这种交易从区块链看起来就像一大朵云。还有一些已知的风险，比如在协调员处泄露数据（这是用户们赖以自动混合资金的服务端）。不过，他认为，技术会继续进步，补上这些缺点；他说，“从 2024 年最后一次讨论以来，攻击界面已经缩小了”。

闪电网络与 eCash

最后，不可小觑的是闪电网络以及 eCash 。基本上，闪电网络是比特币原生的链外交易协议；相比所有链上的隐私性解决方案，它有一个重大好处，是不会在区块链上留下踪迹。因此，理论上更容易实现隐私性。不过，在现实中，依然有许多工作要做，因为绝大部分隐私的使用闪电网络的方法，从用户体验角度来看都很难，需要用户运行自己的闪电节点并自己管理流动性。

虽然市场上有许多易于使用的闪电钱包软件，但绝大部分（就不说全部不算）都需要一定程度的数据共享信任，要与钱包软件公司的服务端分享数据。网络隐私性的一些措施可以缓解风险。

Ecash 也是作为一种强大隐私技术而兴起的， 但在西方国家，采用度依然不高。Fedi 和 Cashu 这样的钱包是最前沿的，它们让用户可以获得比特币世界前所未有的隐私性，代价是要信任托管性质的铸币厂，他们会保管比特币作为 ecash token 的储备。

结论

总的来说，比特币隐私性工具还在继续提升，因为整个行业对这件事的激情从未止息。其中一些比较容易使用，但正如中本聪所展示的，唯有珍视隐私性的人，才能一直留住它。
